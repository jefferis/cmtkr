# cmtkr

## Installation
Currently there isn't a released version on [CRAN](http://cran.r-project.org/).

### Released versions
The recommendation is to install from our lab repository:

```r
install.packages("cmtkr",repos='http://jefferislab.org/R',type='source')
```
